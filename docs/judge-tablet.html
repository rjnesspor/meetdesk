Judge Tablets

This guide will teach you all about how to get your judges started with entering scores.

Setup​

Ensure you have MeetDesk installed and running on the meet director's computer, and make note of the server IP address. Also, prepare whatever device you'd like to use to input scores. Contrary to the title of this page, it doesn't need to be a tablet! Any device with network and browser capabilities will do.

On the scoring device, navigate to:

http://<server-ip>:4721/scoring

This will bring up a login page, where you can select which event and (optional) judge number this device is for. Once chosen, you will be presented with the scoring screen!

Usage​

Once you've set up the scoring device, the hard part is over! All you or your judge have to do to input a score is type in the bib number of the athlete you'd like to score, type in a score, and press Submit Score.

MeetDesk will automatically reject unknown bib numbers or invalid scores.

note

Make sure you provide your judges with score cards with bib numbers, since MeetDesk currently cannot search by athlete name.

Scores entered will be saved to MeetDesk's database and be visible from the meet director's control panel.
