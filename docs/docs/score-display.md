# Score Display

This guide will teach you all about how to set up a score display to start displaying incoming scores!

## Setup

Ensure you have MeetDesk installed and running on the meet director's computer, and make note of the server IP address. Also, prepare whatever device you'd like to use to display scores. Devices with large screens are recommended (TV, projector, etc.) but any device with browser and network capabilities will do.

On the display device, navigate to:

`http://<server-ip>:4721/display`

This will bring up a setup page, where you can select which event(s) you'd like to display, along with an option to configure the display duration. Press `Launch Display` to finish setup.

## Usage

Once you've set up the display device, you're done! MeetDesk will automatically display *any* entered scores for the event(s) you chose when they come in.

:::info
Huge influx of scores coming in? No problem! MeetDesk's display has a built-in queue ensuring *all* scores get displayed.
